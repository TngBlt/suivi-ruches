# 🐝 Suivi Ruches

Application de suivi apicole avec assistance IA.

## ✨ Fonctionnalités

- 📊 Gestion de ruches (Dadant 8-10 cadres)
- 🧠 Dr. Abeille - Assistant IA (Claude)
- 🌿 Analyse Flore & Faune
- 📸 Gestion de photos
- 📅 Journal d'interventions
- 📋 Inventaire matériel
- 💾 Stockage localStorage

## 🚀 Installation locale

```bash
npm install
export ANTHROPIC_API_KEY="sk-ant-..."
npm start
```

Accédez à http://localhost:3000
